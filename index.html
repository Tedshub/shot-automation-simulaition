📝 Cara Menggunakan Visualisasi

  • Pilih MODE MANUAL atau MODE AUTO dengan tombol terpisah
  • Pada MODE MANUAL: Gunakan joystick KP untuk menggerakkan crosshair
  • Pada MODE AUTO: Klik area kamera untuk set target, crosshair akan mengikuti dengan delay 1 detik
  • Adjust parameter KP dengan menggerakkan joystick KP Control
  • Klik tahapan algoritma untuk melihat penjelasan detail
  • Perhatikan bagaimana servo bergerak mengikuti pergerakan crosshair
  • Lihat perubahan real-time pada koordinat dan servo positions
